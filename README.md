## `node` 版本推荐

面向现代，所以推荐大家使用 `node` 当前的长期维护版本 `v18`

## `rust` 版本推荐

最新的稳定版本即可


## 开发

前端建议使用 `pnpm` 包管理器

1. 安装前端依赖

```shell
pnpm install
```

2. 运行开发环境

```shell
cargo tauri dev
```
